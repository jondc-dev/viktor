Set up automated workspace backup system:

### Components
- **Backup script:** `~/clawd/scripts/backup-to-drive.sh`
  - Zips workspace excluding large files (.mp3, .ogg, .wav, .zip, .git, venv, __pycache__)
  - Uploads to `gdrive:***Important Backups***/viktor-backups/`
  - Keeps last 3 local copies
  
- **Restore script:** `~/clawd/scripts/restore-from-drive.sh`
  - Lists available backups: `./restore-from-drive.sh`
  - Restore latest: `./restore-from-drive.sh latest`
  - Restore specific: `./restore-from-drive.sh clawd-backup-YYYY-MM-DD_HH-MM-SS.zip`
  - Creates pre-restore backup in `/tmp` for safety

- **Cron job:** `gdrive-backup` (id: 272a7552-9153-4cc4-9801-cb0c4773e8a7)
  - Schedule: Every 4 hours (0 */4 * * *)
  - 6 backups per day

### rclone Config
- Remote name: `gdrive`
- Config at: `~/.config/rclone/rclone.conf`
- OAuth credentials in: `~/.config/gog/client_secret.json`
