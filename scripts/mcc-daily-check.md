# MCC Daily Check Protocol

## Schedule
- 08:00 AM
- 12:00 PM
- 04:00 PM
- 08:00 PM

## Tasks

### 1. Rufus AI Dashboard
- Login to mcc.saniservice.com
- Go to Marketing HQ → Dashboard
- Check AI-generated insights and focus areas
- Note any new recommendations

### 2. AdIntel Autopilot
- Go to AdIntel → AI Autopilot
- Review recent AI decisions
- Check campaign status changes
- Note any paused/activated campaigns

### 3. At-Risk Customer Segments
- Go to Predictive AI → Customers
- Check RFM segments for at-risk count
- Compare to previous check
- Flag significant changes (>5% movement)

## Reporting
- Send WhatsApp summary to JV only if:
  - Urgent items found
  - Significant changes in at-risk customers
  - Campaigns paused/activated unexpectedly
- Otherwise log to daily memory file
